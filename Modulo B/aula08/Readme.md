----------- Operadores Aritméticos --------- 
-Há precedencia dos operadores ("()", "**", "*, / e %", "+ e -")
"+" soma
"-" subtração
"*" multiplicação
"/" diivisão
"%" resto
"**" exponenciação


----------- Operadores Relacionais --------- 
-não há ordem de precedência.

">" maior
"<" menor
">=" maior igual
"<=" menor igual
"==" igualdade
"===" igualdade restrita
"!=" diferença


----------- Operadores Lógicos ------------
-Prioridade de execução ! -> && ->  e ||.

"!" negação
"&&" conjunção
"||" disjunção


----------- Operadores Ternário --------- 
-Estrutura: teste ? true : false

Ex.: média >= 7 ? "Aprovado" : "Reprovado"
